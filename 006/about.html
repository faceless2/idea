Field Qualifiers to Selects

Change the radio button groups ("certain", "uncertain" etc) when editing fields to a SELECT. Why? Several reasons:

  • With the new proposed design, the "grouping" of the field label, the field, and the radio buttons below it is not as clear as it might be
  • Fields without radio buttons are unevenly spaced
  • Without this we have very long field boxes - too long for some fields. Somehow this now seems less important
  • Radio buttons can't be deselected! I've always found this a problem on the old site too - once a field is marked as "confident" I can never return it to "unspecified confidence". With this design, you can do this - I think that's a good thing, but if it's not desirable it can easily be changed (to disallow the selection of the blank option)

To make this work visually I had to change "intentionally blank but still living" to "still living".
